Специальные символы

Сайт на раздел HTML Symbols

HTML Symbol Entities (или просто HTML Entities) представляют собой специальные коды, используемые в HTML для отображения символов, которые имеют специальное значение в HTML или которые трудно ввести с клавиатуры. Эти коды начинаются с амперсанда (&) и заканчиваются точкой с запятой (;). Они позволяют включать в HTML текст такие символы, как знаки препинания, математические символы, символы валют и многие другие.
