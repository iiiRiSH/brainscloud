HTML-таблицы

HTML-файл

  1. <table></table> - создаём таблицу
  2. В теги <table> прописываем class= "table"
  3. Для симантики внутри тега table вводим теги
    <thead></thead> - голова
    <tbody></tbody> - тело
    <tfoot></tfoot> - подвал
  4. Далее создаем ячейки
    <tr> - строка
    <td> - ячейка
    <th> - главная ячейка
    атрибут colspan="4" - объединяет ячейки, в данном случаи объядинил 4 ячейки

CSS-файл

  • width: 100%; - ширина таблицы на всю ширину сайта
  • border: 1px solid black; - граница таблицы и ее толщина, тип (сплошная) и цвет
    (также типы бывают dashed(граница черточками), dotted (точками))
  • border-collapse: collapse; - схлопывает границы делая единой
  • table-layout: fixed; - делает ячейки равными по размеру
  • padding: 5px 10px 8px 15px; - отступы (по часовой стрелки: верх, право, вниз, лево)
