HTML-теги форматирования текста

Тег для заголовка

<h1> </h1> - заголовок первого уровня, используется один раз,
имеет самый большой шрифт, самое главное значение

<h2> </h2>

<h3> </h3>

<h4> </h4>

<h5> </h5> - редко используется

<h6> </h6> - редко используется

Обычно на практике, максимум используется до четвертого уровня h4. От h1 шрифт уменьшается.

Тег для абзаца (парагрвф)

<p> </p> - в этот тег можно обернуть любой текст

<br> - тег для переноса строки

<b> </b> - тег для жирного шрифта

Тег <strong> </strong> визуально работает также как тег <b>,
но по смыловому значению делает акцент на выделенном участке для поисковых роботов

Пример:Текст

<i> </i> - тег для наклонного шрифта

Тег <em> </em> визуально работает также как тег <i>,
но по смыловому значению делает акцент на выделенном участке для поисковых роботов

Пример:Текст

<u> </u> - тег для подчеркнутого шрифта

Пример:Текст

<s> </s> - тег для перечеркнутого шрифта

Пример:Текст

Теги для списка

  1. <ul>
    <li>элемент списка</li>
    </ul> - маркированный список

    Пример:

    Список продуктов

    • Чай
    • Мороженое
    • Фрукты

  2. <ol>
    <li>элемент списка</li>
    </ol> - нумерованный список

    Пример:

    Рецепт приготовления жареной картошки

    1. Почистить картошку, лук
    2. Порезать картошку, лук
    3. Разогреть масло и жарить

  3. <dl>
    <dt>Заголовок</dt>
    <dd>Описание заголовка</dd>
    </dl> - список с подробным описанием     (редко используемый)

    Пример:

    Заголовок
    Описание заголовка
    Заголовок 2
    Описание заголовка 2
