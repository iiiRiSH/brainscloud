Свойства для стилизации текста

Пишем CSS код:

  1. В папке проекта создаем css-файл. Называем файл style.css (это часто используемое название, но в принципе оно может быть любое)
  2. Далее делаем связку, подключаем css-файл при помощи тега <link> (вводим его в файле index.html в теги <head> перед тегом <title>)
    <link rel="stylesheet" href="/style.css">
  3. Пишем первые стили. Тег <body> - в нем пишутся стандартные стили для всего документа.

    body {
    font-family: Arial, sans-serif; - стиль шрифта, второй шрифт указывается если не сработал первый
    font-size: 15px; - размер шрифта
    line-height: 18px; - междустрочный отступ или можно указать относительное значение 1,5 напр, (15px * 1.5 = 22.5px)
    color: black;
    }

  4. Прописываем class для Заголовка.

    В html-файле в теги h1 вводим class
    <h1 class="title">Заголовок</h1>

    Затем в css-файле вводим название class и прописываем свойства и значения для Заголовка
    .title {
    text-align: center; - выравнивание текста по центру, left по левому краю, right - по правому краю
    letter-spacing: 2px; - отступ между букв (может быть отрицательное значение -2 напр, тогда буквы будут слипаться)
    text-decoration: underline; - подчеркивает текст, в нашем случаи заголовок, overline - подчеркивание сверху, line-through - перечеркивание текста
    text-transform: uppercase; - делает все буквы заглавными, lowercase - делает все буквы маленькими
    }

    Значение justify придаёт тексту равномерное распределение

    .имя class {
    text-align: justify; - пример равномерного текста ниже
    font-weight: normal; - вес шрифта, нормальный. Bolt - жирный шрифт.
    font-style: italic; - наклонный стиль шрифта, normal - обычный
    text-transform: capitalize; - делает первую букву каждого слова заглавной
    }

    Создаём отступ для абзаца
    p {
    text-indent: 20px;
    }

Свойства списка

Маркированный список

  • Первый
  • Второй
  • Третий

ul {
list-style-type: square;
}

Нумерованный список

  1. Первый
  2. Второй
  3. Третий

ol {
list-style-type: decimal-leading-zero;
}

Позиция элементов списка

ul {
list-style-position: inside; - список сдвигается, маркеры помещаются внутрь нашего списка
}
outside, элементы выйдут за пределы списка
