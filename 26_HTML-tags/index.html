HTML-теги (abbr, code, cite, blockquote и др.)

Тег <small></small>

В HTML тег <small> используется для обозначения текста или фрагмента текста, который представляет собой второстепенную информацию или примечание. Этот тег предназначен для стилизации текста таким образом, чтобы он был меньшего размера по сравнению с окружающим текстом.

Пример
290 .00

Тег <sup></sup>

В HTML тег <sup> используется для создания надстрочного текста, который обычно используется для указания примечаний или верхних индексов.

Пример
290 .00 руб

Тег <sub></sub>

В HTML тег <sub> используется для создания подстрочного текста, который обычно отображается ниже базовой линии и меньшего размера шрифта. Это полезно, например, для записи химических формул, математических выражений или сокращений.

Пример
290 .00 руб

Тег <code></code>

В HTML тег <code> используется для обозначения фрагментов компьютерного кода или текста, который представляет программный код. Этот тег позволяет браузеру отображать текст внутри него моноширинным шрифтом (обычно это шрифт, в котором каждый символ имеет одинаковую ширину), что делает код более легким для чтения и отличает его от обычного текста.

Пример
div

Тег <abbr></abbr>

В HTML тег <abbr> используется для определения аббревиатуры или акронима. Он помогает браузерам и другим программам понять, что определенный текст является сокращением, и предоставляет расширенную информацию при наведении курсора или при запросе.

title="HyperText Markup Language" — атрибут title задает расширенную подсказку, которая появляется, когда пользователь наводит курсор на сокращение.

Пример
HTML

Тег <q></q>

Тег <q> в HTML используется для обозначения коротких цитат. Он оборачивает текст, который представляет собой цитату, и по умолчанию браузеры отображают его в кавычках.

Пример
Сложнее всего начать действовать, все остальное зависит только от упорства.
Амелия Эрхарт

Теги <blockquote></blockquote>
<cite></cite>

В HTML тег <blockquote> используется для обозначения длинных цитат, которые обычно выделяются в тексте для придания им особого значения. Этот тег предназначен для выделения блоков текста, содержащих цитаты из другого источника. В большинстве браузеров текст, заключенный в тег <blockquote>, автоматически получает отступы.

Тег <cite> в HTML используется для обозначения названия произведения, книги, статьи, фильма, песни или другого творческого произведения. Обычно его содержимое отображается курсивом в браузере по умолчанию. Этот тег помогает семантически отметить источники и цитаты, что может быть полезно для поисковых систем и других инструментов анализа контента.

Пример

«Никогда не поздно или — в моём случае — никогда не рано стать тем, кем ты хочешь стать. Временных рамок нет, можешь начать когда угодно. Можешь измениться или остаться прежним — правил не существует»

Фрэнсис Скотт Фицджеральд

Теги <del></del>
<ins></ins>

В HTML тег <del> используется для обозначения текста, который был удален из документа. Обычно он отображается зачеркнутым, чтобы показать, что он больше не является частью текущей версии документа.

Тег <ins> в HTML используется для обозначения вставленного текста. Этот тег часто применяется в сочетании с тегом <del>, который обозначает удаленный текст. Использование этих тегов помогает визуально показать изменения в документе, например, добавление или удаление текста в правках.

Пример
Старое/неверное действие Новое/верное действие

Теги <dfn></dfn>

Тег <dfn> (от англ. definition — определение) используется в HTML для выделения термина, который определяется в окружающем тексте. Он помогает улучшить семантику документа и облегчает понимание контекста. Обычно <dfn> используется вместе с тегом <abbr> (если термин является аббревиатурой) или <strong> (если нужно выделить важный текст).

Пример
Термин — слово или словосочетание, точно и однозначно именующее понятие и его соотношение с другими понятиями в пределах специальной сферы.

Теги <kbd></kbd>

Тег <kbd> в HTML используется для обозначения текстового ввода с клавиатуры. Он обычно применяется для отображения клавиш или комбинаций клавиш, которые пользователю нужно нажать, например, в инструкциях или документации.

Пример
ctrl + s - сохранить
