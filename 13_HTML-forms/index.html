Тег <label></label>

Тег label предназначен для обозначения что в данное поле вводить,
для привязки необходимо в атрибут for="" ввести уникальный id="" input

Свойство css display: block; позволит разместить название строки сверху

Тег <select></select>

Позволяет выбрать нужный вариант из предложенного выходящего списка

Тег input с атрибутом type="radio"

  • Атрибут name="" должени быть одинаковым, так как позволяет выбрать только один вариант из предложенных.
  • Атрибут id="" всегда уникальный.

Тег input с атрибутом type="checkbox"

Тег <button></button> - для создания кнопки

  • Атрибут type="submit" позволяет при нажатии на кнопку - отправить заполненную форму
  • Атрибут type="reset" позволяет при нажатии на кнопку - очистить заполненную форму

Тег input с атрибутом type="file"

Этот атрибут позволяет загрузить файл, в данном примере это фото.
