Вставляем картинку

Для начала нужно создать папку assets для структуризации, в ней будут хранится все необходимые вложенные элементы для сайта
внутри папки assets создаем папку images и в нее добавляем картинки

Чтобы вставить картинку требуется тег <img> он одиночный
в атрибуте src= указывается путь к картинке
в атрибут alt= можно ввести альтернативное название,
которое будет появляться, если картинка не загрузится, но его можно оставить пустым

Вставить картинку с другого сайта


Картинку расположенную на другом сайте, необходимо вставить на наш сайт
Правой кнопкой мыши щелкаем по картинке и выбираем Копировать адрес изображения
вставляем адрес в наш тег img в атрибут src=, готово

Дополнительные атрибуты для картинки

width="___" ширина картинки, если ввести только параметр ширины, высота изменится пропорционально ширине
и высоту можно вовсе не указывать, но при необходимости высота указывается в атрибуте height="___"

Атрибут title="___" позволяет увидеть название картинки при наведении на нее курсором
Атрибут title="___" является глобальным и его можно использовать для других тегов, например ссылки
