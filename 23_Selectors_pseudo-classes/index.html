Селекторы, псевдоклассы

.text p класс изменит стиль всех вложенных параграфов текста

.text > p класс изменит стиль только "прямых потомков" текстового блока

.list li класс изменит стиль всех вложенных элементов списка

.list li:first-child псевдокласс изменит стиль только первого элемента списка

.list li:last-child псевдокласс изменит стиль только последнего элемента списка

.list li:nth-child(2) псевдокласс изменит стиль того элемента списка что указан в скобках

.list li:nth-child(odd) псевдокласс изменит стиль нечётных элементов списка

.list li:nth-child(even) псевдокласс изменит стиль чётных элементов списка

.text > p:first-of-type псевдокласс изменит стиль первого дочернего элемента по типу p

.table tbody tr:nth-child(even) псевдокласс изменит стиль чётных элементов таблицы
(удобно изменять бэкграунд для лучшей читабельности)
odd - для нечётных строк

.checkbox:checked + .label псевдокласс помогает изменить стиль label после наведения на него курсором

.text span[title] глобальный атрибут(title) помогает добавить дополнительной стилилизации для span
