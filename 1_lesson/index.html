Создание скелета HTML-документа

  1. Создаем папку с названием проекта. Латинскими буквами. Без пробелов.
  2. Открываем папку в VS Code. И создаем первый файл под названием index.html.
  3. В первую строку вводим тип документа.
    <!doctype html>.
  4. Далее вводим тег <html></html> - в ней будет находиться вся html-страница.
  5. В тег <html> вводим теги:
    <head></head> - здесь находится основная информация о сайте (заголовок, кодировка, мета информация, css-стиль).
    <body></body> - здесь отображается всё содержимое сайта.
  6. В тег <head> вводим теги:
    <title>здесь указывается название вкладки которое появляется в браузере</title>.
    И над тегом <title> вводим тег кодировки <meta charset="utf-8">.
    Для корректного отображения страницы HTML, веб - браузер должен знать набор символов (character encoding) для использования.

Если нажать клавишу "!" затем "tab", то выйдет готовый начальный скелет сайта.
