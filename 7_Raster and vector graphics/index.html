Растровая и векторная графика, форматы изображений

Существует два основных вида графики: растровая и векторная. Растровая картинка состоит из множества пикселей. Самые популярные форматы растровой графики: .jpeg (фото, снимок) и .png (растровый объект без фона). .png формат - тяжелый, его следует выбирать только при необходимости использования картинки без фона.

Формат .jreg всегда с фоном Формат .png без фона

При использовании векторной графики мы выбираем .svg формат.
В основном вектор используется для иконок
