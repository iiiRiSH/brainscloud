Псевдоклассы

Псевдоклассы это интерактив с нашими элементами. При наведении курсора на элемент он изменяет свой цвет, бэкграунд и т.п. Элемент становится заметным и показывает готовность к взаимодействию с ним.

Псевдокласс :hover

Добавляем к обычному классу псевдокласс :hover без пробелов и прописываем свойства для создания интерактива.

При наведении курсора на ссылку появляется нижнее подчеркивание.

В данном случае псевдокласс :hover добавляем к классу относящийся к кнопки. В итоге цвет кнопки при наведении на нее курсором изменяется на более темный.

Нет взаимодействия

При наведении курсором на кнопку

Свойство для псевдокласса :hover cursor: pointer

При наведении на кнопку вместо стрелки появляется рука

Псевдокласс :focus

Убрали подсвечивающую границу и при нажатии на input бэкграунд изменил цвет.

Псевдокласс :active

Интерактив осуществляется при нажатии на кнопку и она меняет цвет на более темный либо на абсолютно любой другой.

Псевдокласс :visited

Этот псевдокласс используется для ссылок после посещения которой она меняет цвет. Рекомендуется использовать при условии наличия большого количества ссылок на сайте.

Рекомендуемый порядок использования псевдоклассов

Псевдокласс :disabled

атрибут disabled прописывается в теги button, что делает кнопку неактивной

затем в css-файле придаются примерно следующие свойства (неактивный вид)

Псевдокласс :checked

Атрибут checked вводится в input с типом radio и checkbox. Визуально показывая по умолчанию выбранный вариант.

css-свойства для псевдокласса :checked рассмотрятся в следующих уроках
