HTML5-теги

Семантические теги - это логические теги, которые заменяют дивы (div) и придают им логику.

<header></header> - шапка сайта

<main></main> - содержание сайта

<footer></footer> - подвал сайта

<nav></nav> - навигация сайта (ссылки)

<section></section> - секция сайта
В каждой секции должен быть заголовок любого уровня от h1 до h6!

<article></article> - для списков, записей, статей, новостей сайта и т.п
Важно наличие заголовока!

<time></time> - для даты (напр, создания статьи, поста)

<aside></aside> - боковая панель сайта

<address></address> - контактная информация сайта

<figure></figure> - для размещения картинки (напр, для новостей)
<figcaption></figcaption> - наименование картинки
