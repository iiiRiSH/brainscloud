Что такое CSS

  • CSS (Cascading Style Sheets) - каскадные таблицы стилей
  • Служит для внешнего оформления HTML документа (шрифт, цвет, фон и тд.)
  • CSS файлы имеют расширение .css
  • CSS файл состоит из набора селекторов, свойств и значений к ним
  • CSS файл читается браузером сверху вниз
  • Дочерние элементы наследуют некоторые свойства их родителей
  • Некоторые теги имеют оформление по умолчанию, которое придает им браузер

Стилизация через class

class - глобальный атрибут, можно использовать для стилизации многих элементов

<h1 class='title'>Заголовок</h1>

.title {
свойство: значение;
свойство: значение;
}

Пример

<h1>Заголовок</h1>
Заголовок - вид в браузере
<h1 class='title'>Заголовок</h1>
Заголовок - вид в браузере

.title {
color: red;
}
это прописывается в css файле

Стилизация тегов

Иногда нужно, но если можно, то используем class

<h1>Заголовок</h1>

h1 {
свойство: значение;
свойство: значение;
}

Наследование стилей

  • Дочерние элементы наследуют некоторые стили родительских элементов, если такие стили не были заданы

body {
font-size: 15px;
color: black;
}

p {
font-size: 18px;
}

  • В основном наследуется стиль оформления текста
