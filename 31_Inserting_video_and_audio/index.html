Вставка видео и аудио

Video

Тег <video></video> используется для встраивания видеоконтента на веб-страницу.


Атрибуты для тега video

  • src: Указывает путь к видеофайлу.
  • width и height: Устанавливают размеры видеоплеера.
  • controls: Добавляет стандартные элементы управления видеоплеером (воспроизведение, пауза, громкость и т.д.).
  • loop: Повторяет видео по кругу.
  • muted: Отключает звук у видео.
  • autoplay: Автоматически воспроизводит видео при загрузке страницы. Совместное использование с muted обычно рекомендуется для предотвращения автоматического воспроизведения звука.
  • preload: Указывает браузеру, как загружать видеофайл при загрузке страницы. Возможные значения:
    • auto: загружает весь видеоконтент.
    • metadata: загружает только метаданные (размер, продолжительность и т.д.).
    • none: не загружает видео до начала воспроизведения.

  • poster: Указывает изображение-заглушку, которое будет отображаться до начала воспроизведения видео.
  • Также можно использовать вложенные теги <source> для указания нескольких форматов видео, чтобы обеспечить поддержку различных браузеров. Этот код добавляет поддержку разных форматов видео и выводит сообщение, если браузер не поддерживает тег <video>.

Audio

HTML-тег <audio> используется для внедрения звуковых файлов в веб-страницы.

Встроить видео с YouTube

Чтобы встроить видео с YouTube на ваш сайт, можно использовать iframe-код, который предоставляет YouTube. Вот пошаговая инструкция:

  1. Найдите нужное видео на YouTube.
  2. Под видео нажмите кнопку "Поделиться" (Share).
  3. Выберите опцию "Встроить" (Embed).
  4. Скопируйте предложенный HTML-код.
Вы также можете настроить размеры iframe, изменив значения width и height атрибутов. Атрибуты allow и allowfullscreen добавляют возможности для воспроизведения видео в полноэкранном режиме и управления другими параметрами.
