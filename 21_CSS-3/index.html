box-sizing: border-box;

Свойство box-sizing: border-box; в CSS изменяет способ расчета размеров элемента.

Без box-sizing: border-box; (по умолчанию - content-box)

При box-sizing: border-box, добавление внутренних отступов (padding) или границ (border) не изменяет размеры элемента, заданные в CSS. Это упрощает управление размерами элементов, особенно при сложной верстке.

Чтобы свойство box-sizing: border-box; применялось ко всем элементам на странице, можно использовать CSS-правило, применяемое ко всем элементам и псевдоэлементам с помощью универсального селектора (*) и селектора псевдоэлементов (*::before и *::after). Обычно это делается в самом начале CSS-файла, чтобы гарантировать, что это правило будет применяться ко всем элементам на странице.

opacity:
непрозрачность

Значение opacity может быть от 0 (полностью прозрачный) до 1 (полностью непрозрачный).
0.5 - полупрозрачный

resize:
растягивание текстового поля (элемента) курсором

  • resize: horizontal; - по горизонтали
  • resize: vertical; - по вертикали
  • resize: both; - как по горизонтали, так и по вертикали
  • resize: none; - невозможность изменения размера элемента

min-height: 150px;
max-height: 250px;

Следующими свойствами мы зададим минимальное и максимальное растягивание элемента по вертикали (высоту).

transition:
плавное изменение свойства

Свойство transition в CSS позволяет анимировать изменения свойств элемента при взаимодействии с ним. Это означает, что когда определенное свойство изменяется, анимация плавно переходит от начального значения к конечному.

  • background-color: Свойство CSS, которое изменяется. В данном случае это цвет фона элемента.
  • 0.2s: Длительность перехода. В этом случае анимация длится 0.2 секунды (200 миллисекунд).
  • linear: Функция временного распределения. Линейная функция означает, что анимация будет происходить с постоянной скоростью от начала до конца.
