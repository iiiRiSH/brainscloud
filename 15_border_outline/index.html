Свойство border и его значения

Общая запись для свойства border (граница) можем иметь вид одной строкой, в которой по порядку указываем ширину, тип(стиль) и цвет. Так и для каждой из сторон отдельно (в примере верхнюю границу убрали записью border-top: none;)

outline

Главное отличие outline от border заключается в том, что мы можем задать обводку для элемента не влияя на размер этого элемента. При увеличении ширины обводки она выходит за пределы элемента.

Синтаксис для outline выглядит также как и для border.

Свойство outline-offset: задаёт отступ от элемента для которого задаётся обводка. Также для отступа может задаваться отрицательное значение, в этом случаи обводка будет заходить на территорию элемента (но смысла в таком решении особо нет).

Outline не может задаваться для каждой из сторон в отдельности. Он устанавливается по умолчанию при создании инпута, текстового поля, но его можно убрать или редактировать.
