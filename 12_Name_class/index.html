Наименование классов

Первый проверочный вопрос для наименования class
Что это за блок?

Второй проверочный вопрос для наименования class
Что делает данный class?

  1. Есть блок с class="article"
    Article в переводе с английского статья, запись.
    Тем самым мы ответили на вопрос Что это за блок?
  2. Второй class имеет наименование "article-title" - что переводится как Заголовок статьи.
  3. Третий class="article-text" - текст статьи.
  4. class="text-red" - отвечает на вопрос Что делает данный class?
    - перекрашивает текст в красный цвет.

Вывод! название classa должно быть понятно всем, не только вам

Наименование классов должно быть...

  • осмысленным
  • вводиться на английском языке
  • без цифр
  • раделяем слова тире
  • введено маленькими буквами
