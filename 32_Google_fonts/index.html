Google fonts, локальные шрифты

Google Fonts

Google Fonts — это библиотека бесплатных шрифтов, которые можно использовать на вашем сайте.

Oсновные шаги по использованию Google Fonts:

  1. Перейдите на сайт Google Fonts.
  2. Нажмите на вюбранный шрифт, чтобы открыть его страницу.
  3. Нажмите на кнопку Get font.
  4. Далее на кнопку Get embed code. Здесь в левой колонке можно выбрать нужные стили (например, обычный, жирный, курсив и т.д.). В правой скопировать код.
  5. Вставьте скопированный код в <head> вашего HTML-документа.
  6. Используйте шрифт в вашем CSS-файле, указав его в свойстве font-family.

Локальные шрифты Web Fonts

  • Необходимо зайти на сайт Web Fonts и скачать архив выбранного шрифта.
  • Затем создать папку куда переместить выбранные шрифты с форматом .woff и .woff2
  • В css-файле сделать соотвествующую запись где нужно указать путь и формат шрифта, а твкже font-weight: при необходимости.
  • Добавить font-family: установленного шрифта для выбранного блока с текстом.
