Блочные и строчные элементы

<div></div> - тег, который разделяет блоки, он не имеет никого внешнего оформления. Это блочный элемент в который можно помещать другие элементы.
Для div мы можем задавать свойства: ширину, высоту, отступ внутрений и отступ внешний с любых сторон.

Здесь margin задает внешние отступы по отдельности низ, верх, право, лево

Здесь margin задает внешние отступы верх, низ по 10px, и по центру

<span></span> - тег, строчный элемент, который позваляет обернуть часть текста для дальнейшего его редактирования через class="".

Через class="" прописываем свойства
display: block; - строчный элемент делает блочным
display: inline-block; - делает элемент строчно-блочным

Как разместить контент сайта по середине страницы

Весь контент сайта оборачиваем в тег div и через class прописываем свойства
