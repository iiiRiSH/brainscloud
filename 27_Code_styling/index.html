Стилизация кода (HTML, CSS)

Рекомендации по написанию html кода (разметки).

  • Необходимо соблюдать вложенность, иерархию элементов. Кто является родительским элементом, кто дочерним.
  • Вертикально разделять одной пустой строкой на логические блоки (элементы).
  • Также можно разделять блоки структурно, двумя пустыми строками, потом прописать комментарий и далее идет блок с кодом тега, например, <header></header>.
  • Если блок вмещает в себя много контента, то закрывающий тег желательно отметить закрывающимся комментарием.
  • Прописываем class первым атрибутом в строке, чтобы он не терялся среди других атрибутов.

Рекомендации по написанию css кода.

  • Необходимо соблюдать упорядочность в написании селекторов и свойств со значениями к ним, использовать пробелы и отступы.
  • Разбиение свойств на группы пустыми строками:
    1 группа - размер и отступы блока ( display, width, height, margin, padding );
    2 группа - всё что связано со шрифтом ( font-family, font-size, color, font-weight, font-style );
    3 группа - внешнее оформление элемента: фон, граница, скругление, тени и т.д. ( background-color, box-shadow, border-radius и т.д. );
    4 группа - все что относится к позиционированию;
    5 группа - transition, можно вынести в отдельную группу.
  • Для значения 0 не прописываем пиксели.
  • Писать цвет сокращённым, если есть такая возможность ( #fff, #ccc, #000 )
  • Там где есть значение с перечислением параметров - ставим запятую и пробел.
  • Прописывать комментарий для стилилизации каждого блока, элемента.
  • Между селекторами оставляем одну пустую строку, чтобы код хорошо читался.
