Свойства flexbox

display: flex;

display: flex; - это CSS-свойство, которое позволяет использовать flexbox-модель для создания гибких и адаптивных макетов. Flexbox (или гибкая коробка) упрощает выравнивание элементов и распределение пространства внутри контейнера, даже когда размер контейнера неизвестен или изменяется.

flex-direction:

flex-direction: определяет направление основной оси (например, row, row-reverse, column, column-reverse).

justify-content:

justify-content: выравнивает элементы вдоль основной оси (например, flex-start, flex-end, center, space-between, space-around, space-evenly).

align-items:

align-items: выравнивает элементы вдоль поперечной оси (например, flex-start, flex-end, center, baseline, stretch).

flex-basis:

flex-basis: flex-basis: задает начальный размер элемента до распределения свободного пространства.

flex-grow:

flex-grow: определяет, насколько элемент может расти относительно других элементов внутри контейнера.
